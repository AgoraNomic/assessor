Amend rule 2659 (Stamps) by replacing:

       Any player CAN, once per week, pay X Stamps, where each specified
       Stamp is a different type, to gain (X^2)-X radiance.

with:

       Any player CAN, once per week, pay X stamps, where each specified
       stamp is a different type, to gain (X-1)*3 radiance.
